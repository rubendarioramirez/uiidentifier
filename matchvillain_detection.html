Shape Detection — matchvillain.png

Two approaches compared: Edge-based (Canny) vs. Color-based (HSV segmentation)

Why did the first image (image.jpeg) miss shapes? Canny edge detection relies on sharp gradient jumps between neighboring pixels. The first screenshot had heavy gradients, anti-aliasing, transparency, and overlapping UI layers — causing edges to fragment. Broken edges → broken contours → missed shapes. This match-3 game has much more distinct color blocks, so both approaches work better here. The color-based approach (Step 4) is particularly effective because it groups pixels by hue rather than hunting for edges.
